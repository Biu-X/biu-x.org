# `biu-x.org`子域名申请
## 申请流程

- Fork 本项目
- 域名信息添加  
  在项目下[zones](./zones)的`biu-x.org`下，按照`#NAME, TYPE, CONTENT`格式添加，一行一条记录
- 提交 Pull Request
- 等待审核，确认后部署
